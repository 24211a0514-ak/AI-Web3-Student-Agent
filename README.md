# AI Powered Web3 Student Support Agent

## Problem Statement
Students often struggle to find reliable information about hackathons, clubs, internships, and learning resources in one place.

## Solution
Our project provides an AI-powered student support agent that helps students discover opportunities and resources through a chatbot interface.

## Features
- AI chatbot for student queries
- Dashboard with Hackathons, Clubs, Resources
- MetaMask wallet integration

## Tech Stack
HTML
CSS
JavaScript
MetaMask

## How to Run
1. Open project in VS Code
2. Run index.html using Live Server
3. Connect MetaMask wallet
